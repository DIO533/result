# 并联式混合动力汽车能量管理策略仿真程序

## 程序说明

本仿真程序基于 **MATLAB 纯脚本**（无需 Simulink）实现了并联式混合动力汽车（HEV）的能量管理策略仿真，对应论文：**2223865 朱晨吉 - 并联式HEV能量管理策略仿真与对比分析**。

---

## 文件结构

```
main_simulation.m              ← 主程序入口（直接运行此文件）
params/
  vehicle_params.m             ← 整车参数（质量、风阻、传动比等）
  engine_params.m              ← 发动机参数（万有特性MAP、功率等）
  motor_params.m               ← 电机参数（效率MAP、最大扭矩等）
  battery_params.m             ← 电池参数（容量、内阻、OCV-SOC等）
models/
  engine_model.m               ← 发动机燃油消耗率计算模型
  motor_model.m                ← 电机效率与电功率计算模型
  battery_model.m              ← 电池SOC等效电路模型（安时积分法）
  vehicle_dynamics.m           ← 整车纵向动力学模型
  transmission_model.m         ← 传动系统（换挡、转速/扭矩折算）
strategies/
  rule_based_strategy.m        ← 基于规则的能量管理策略（逻辑门限）
  fuzzy_logic_strategy.m       ← 模糊逻辑控制策略
  ecms_strategy.m              ← 等效燃油消耗最小化策略（ECMS）
driving_cycles/
  generate_NEDC.m              ← NEDC工况生成（1180s，~11km）
  generate_WLTC.m              ← WLTC工况生成（1800s，~23km）
analysis/
  compare_strategies.m         ← 跨工况策略对比分析
  single_factor_analysis.m     ← 单因素变量法分析
  plot_results.m               ← 图表绘制（7类图表）
  sensitivity_analysis.m       ← 敏感性分析
results/                       ← 仿真结果输出目录（自动创建）
README_simulation.md           ← 本说明文档
```

---

## 快速开始

1. 打开 MATLAB（建议 R2018a 及以上版本）
2. 将整个项目文件夹添加到 MATLAB 路径，或直接进入项目根目录
3. 在命令窗口运行：
   ```matlab
   main_simulation
   ```
4. 程序自动完成所有仿真并将图表保存到 `results/` 文件夹

**预计运行时间**：约 2~5 分钟（取决于计算机配置）

## 仿真内容

### 整车参数（基于丰田普锐斯级别）

| 参数 | 数值 |
|------|------|
| 整车质量 | 1500 kg |
| 发动机峰值功率 | 57 kW |
| 电机峰值功率 | 50 kW |
| 电池容量 | 6.5 Ah / 201.6 V |
| 风阻系数 | 0.26 |
| 迎风面积 | 2.33 m² |

### 工况对比

| 工况 | 时长 | 距离 | 最高车速 | 特点 |
|------|------|------|----------|------|
| NEDC | 1180 s | ~11 km | 120 km/h | 城市+郊区，低动态 |
| WLTC | 1800 s | ~23 km | 131 km/h | 四阶段，高动态 |

### 能量管理策略

| 策略 | 原理 | 特点 |
|------|------|------|
| 基于规则（Rule-Based） | SOC门限+扭矩阈值判断 | 简单可靠，次优 |
| 模糊逻辑（Fuzzy） | 5×4模糊推理规则表 | 鲁棒性好，平滑 |
| ECMS | 等效燃油最小化瞬时优化 | 最优（局部最优接近全局） |

## 模型说明

### 发动机模型
- 基于 **11×13 BSFC MAP**（万有特性曲线）进行双线性插值
- 输出：燃油消耗率 [g/s]、制动比油耗 [g/kWh]

### 电机模型
- 基于 **14×10 效率MAP**（含驱动和制动象限）
- 考虑恒扭矩区/恒功率区的转矩限制

### 电池模型（等效电路）
- **OCV-SOC** 查表 + 内阻压降模型
- 采用安时积分法更新 SOC
- SOC 保护范围：0.2 ~ 0.9

### 传动系统
- 5挡自动变速箱（基于车速的换挡策略）
- 并联式结构：发动机与电机通过离合器连接，同轴布置

## 参数修改指南

如需修改仿真参数，请编辑对应的参数文件：

- **初始SOC**：修改 `params/battery_params.m` 中的 `bat.SOC_init`
- **ECMS等效因子**：修改 `params/battery_params.m` 中的 `bat.s_factor`
- **发动机启停阈值**：修改 `params/engine_params.m` 中的 `eng.soc_start / eng.soc_stop`
- **电机参与比例**：修改 `params/motor_params.m` 中的 `mot.power_ratio`

## 注意事项

1. 首次运行前请确保 MATLAB 路径包含所有子文件夹，或在项目根目录下运行
2. 图表以 `.fig`（MATLAB格式）和 `.png`（通用格式）双格式保存
3. 所有图表窗口为不可见模式（`Visible: off`），直接保存文件，不会弹出大量窗口
4. 若需修改工况数据，可编辑 `driving_cycles/` 下的对应文件
5. 代码具有完整的中文注释，便于理解和修改
